  2. What is the selector used to find paragraphs adjacent to an h1?
  3. What is the selector used to find paragraphs indirect siblings to an h1?
  4. Find at least two selectors that can find the section element with the class name article-content.
  5. Find the span that contains "world" as one of two space delimited items in the "data-test" attribute.
  6. Find the first child and the last child in this list with the "test-answers" id (think commas).
  7. Find every odd entry in this list of "test-answers".
  8. Find elements with a class attribute of any value.
  9. Find elements with a data-test attribute beginning with "hello".
  10. Find elements with a data-test attribute that ends exactly with "world".
  11. Find elements with an id that contains the substring "block-header".
  12. Find elements with an id that contains a hyphenated list of values that begins with "test".
  13. Compare the following: #test-answers :nth-child(2) and #test-answers :nth-last-child(2).
  14. Compare the following: #test-answers li:nth-of-type(2) and #test-answers li:nth-last-of-type(2).
  15. Find all of the empty tags on the page (e.g. input, img, etc).
  16. Find anchor tags that are hyperlinks of some kind (clickable).
  17. Find elements that are declared to be in "en"glish in the document (note: this uses a specific pseudo-class selector). Understand why the entire document is considered selected.
  18. Find all p tags that are, by declaration or inheritence, in "en"glish.
  19. Find all enabled input elements on the page.
